# ~~中~~(X)清華大學校務系統 - 課程歷年成績資料下載器

# NTHU Course Grade Data Downloader

---

## 前置要求

安裝[node.js](https://nodejs.org/zh-tw/download/)（建議版本 18 以上）。

使用TypeScript編寫。

---
## 使用說明

整個資料夾下載下來後，

登入校務系統，從網址獲取你的ACIXSTORE值。

將ACIXSTORE值用雙引號包起來，放在`NTHU.ts`的開頭處：

```typescript
const ACIXSTORE = "你的ACIXSTORE值";
```
開啟命令提示字元或終端機，`cd`進入專案資料夾後

依序執行以下指令：

`npm i`

`npx tsc`

`npm start` (或 `node main.js`)

若沒有在檔案設定你的ACIXSTORE，

則執行後，會先讓你貼上你的ACIXSTORE值，

然後會再詢問要下載哪一學年、哪一學期的課程資料。

可使用鍵盤上/下鍵來選擇選項。

輸入後按下 Enter 即可開始下載。檔案較大，請~~忍一下~~耐心等待。

---

## 下載結果

會存到 `data` 資料夾中 ~~但我有上傳 所以你直接去那裏面載檔案也可以~~

檔名格式為 `NTHU_學年_學期.html`，學年為民國年，學期部分：10 代表上學期，20 代表下學期。
