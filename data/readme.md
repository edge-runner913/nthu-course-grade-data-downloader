## 近5年課程平均分數&標準差檔案

存有109~114年的歷年上/下學期成績。另附上108之檔案(但無成績資料。)

檔名格式為 `NTHU_學年_學期.html`，學年為民國年；`1` 代表上學期，`2` 代表下學期。

`.html`檔案下載下來，在瀏覽器打開即可閱覽。可自行利用 `F3` 或 `Ctrl + F` 搜尋科目。

預設按科號排序。

>[!WARNING]
>最新學年(114_1)的成績尚未完全上傳完畢，隨時可能更新，
>
>如需要最新成績，可自行查詢，或運行腳本自行爬取。
